# English Makase - Product Requirements

This document outlines the core rules and features for the "English Makase" application.

1. **App's default language is Korean.**
2. **This app shows today's vocabulary that is related to a personal hobby.**
3. **User chooses their hobby boundary at the first time when they open the app.**
4. **Hobby boundary is saved in local storage.**
5. **Every day, when the user opens the app, the app searches a pre-compiled local database for sentences matching the user's hobby boundary and picks 1~3 Today's sentences.**
6. **The app selects today's sentences by three rules:**
   - 6-1. It should contain new words or expressions that the user never saw (tracked locally).
   - 6-2. It should contain unusual expressions but they must be commonly used in real life (e.g., native idioms).
   - 6-3. It should be related to the user's hobby or interest.
7. **The app has a history function that shows past days' sentences.**
